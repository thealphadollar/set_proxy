Proxy Master

One script to set it all!

Download

set_proxy.sh

Instructions

Set the proxy

Open the downloads directory in the terminal.

Execute the following:

$ sudo bash set_proxy.sh -h <proxy_host> -p <proxy_port>

For IIT KGP users, enter proxy host -> 172.16.2.30 and proxy port -> 8080

Unset the proxy

Open the downloads directory in the terminal.

Execute the following:

$ sudo bash set_proxy.sh -u

Known Issues

Some Users might need to change Settings->Network->Proxy from Off to Manual.
